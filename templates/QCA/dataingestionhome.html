Establish connection

Establish secure and reusable connections to various data sources like databases, file systems, and cloud services. These connections enable seamless data and metadata extraction and can be reused in the DQ configuration section for consistent and efficient data management. By utilizing these pre-configured connections, you ensure secure and optimized access to your data resources across different platforms.

Ingestion

***Ingestion is the process of collecting, importing, and processing data from various sources into a storage system or data platform for further analysis. It is a crucial step in the data pipeline, ensuring that raw data is extracted from disparate sources (such as databases, APIs, logs, or files) and made available for processing, transformation, and analysis.
